# Java Swing Snake Game

A classic Snake game built using Java Swing. Guide the snake to eat the apples, grow larger, and see how high a score you can achieve!

## Prerequisites

To run this game on your local device, you will need:
- Java Development Kit (JDK) 8 or higher installed on your system.
- Ability to run commands from your terminal/command prompt.

## How to Run Locally

Follow these instructions to compile and play the game:

1. **Open your terminal or command prompt.**
2. **Navigate to the game directory.** Path where the game files are located:
   ```bash
   cd "/Users/"your username"/Swing Application"
   ```
3. **Compile the Java files.** This will convert the `.java` files into executable `.class` files:
   ```bash
   javac *.java
   ```
4. **Run the Application!** Start the game by executing the main class:
   ```bash
   java SnakeGame
   ```

## Controls

- Use your **Arrow Keys** (Up, Down, Left, Right) to direct the Snake.
- If you lose, click the **Restart** button to try again!
